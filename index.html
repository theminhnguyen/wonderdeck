Welche Bilder verwenden? 🖼️

Die Tiefe entsteht durch mehrere Ebenen pro Folie. Ein Bild ziehst du einfach auf die Bühne — jede Ebene kannst du rechts einstellen.

Ebenen-Typen

Hintergrund Ganzflächiges Bild (Himmel, Landschaft, Farbverlauf). JPG oder PNG, ohne Transparenz. Wenig Parallax (z. B. 10–16).
Vordergrund / Motiv Ein freigestelltes PNG mit transparentem Hintergrund (Person, Berg, Objekt). Nur so sieht man den Hintergrund dahinter → echte Tiefe. Mehr Parallax (z. B. 35–55).
Reaktive Deko Transparentes PNG an den Rändern (z. B. Blätter). Schalter „reaktiv" an → weicht im Präsentationsmodus vor dem Cursor zurück.

Format & Größe

  • PNG = mit Transparenz (für freigestellte Motive & Deko). JPG = kleiner, kein Transparenz (für Hintergründe).
  • Mindestens ~1920 px breit, besser 2000 px+ — damit es im Vollbild scharf bleibt.
  • Seitenverhältnis möglichst 16:9 für Hintergründe.

Pro Stil

  • ✦ Wonder: 1 Hintergrund + optional ein freigestelltes Motiv + reaktive Deko an den Ecken. Starker Intro-Zoom.
  • ▦ Snap: Hintergrund + 1–2 freigestellte Ebenen (fern/nah) für Tiefe. Ideal für Story-Folien.

Damit Text lesbar bleibt

  • Dunkle Bildränder/Vignette helfen. Setze Text auf die ruhigere Bildhälfte.
  • Position, Breite & Ausrichtung jedes Textes stellst du rechts ein.

Woher Bilder nehmen?

  • KI-Bilder: z. B. „Nano Banana" (Gemini) — gratis.
  • Freistellen / in Ebenen schneiden: Photopea (photopea.com, gratis, im Browser).
  • Oder eigene Fotos / freie Stock-Bilder.

💡 Tipp: Über „Backup" oben kannst du deine Präsi jederzeit als Datei sichern.

Beispiele ✨

Lade eine fertige Präsentation als Startpunkt. Sie wird als neue Präsi angelegt und ersetzt die aktuelle Ansicht (vorher ggf. über „Backup" sichern).

Neue Folie · Layout wählen

Wähle ein Layout als Startpunkt. Bilder ziehst du danach einfach auf die Bühne.

Meine Präsentationen 📑

Wechsle zwischen Präsentationen, benenne um oder lösche. Nav-Einträge können auf eine andere Präsentation verweisen.
