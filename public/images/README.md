# public/images/ — Bild-Assets für reference.html

Lege hier deine echten Bilder ab, dann ersetze in `reference.html` die
CSS-Platzhalter (markiert mit `▶ ECHTES BG-BILD` bzw. `▶ ECHTE FIGUR`).

Erwartete Dateien (Beispiel-Namensschema):

| Sektion | Hintergrund        | Vordergrund / Motiv    |
| ------- | ------------------ | ---------------------- |
| 1       | `hero1-bg.png`     | `hero1-figure.png`     |
| 2       | `hero2-bg.png`     | `hero2-figure.png`     |
| 3       | `hero3-bg.png`     | `hero3-figure.png`     |

**Empfehlung:** PNG/JPG, mind. ~2000 px Breite, 16:9 für Hintergründe;
freigestellte Motive (Figuren) als **PNG mit transparentem Hintergrund**.

> Hinweis: WonderDeck selbst lädt KEINE Dateien aus diesem Ordner — dort
> kommen die Bilder per Drag-and-drop rein. Dieser Ordner ist nur für die
> eigenständige `reference.html`.
